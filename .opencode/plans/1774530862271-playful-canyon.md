# Notification Handler Implementation Plan

## Overview
Implement a DBus notification handler for Quickshell with:
1. **Toast notifications** - Temporary popups that auto-dismiss
2. **Notification center** - Persistent list to review notification history

## Files to Create

### 1. `shell/notifications/NotificationServer.qml` (service singleton)
- Instantiate Quickshell's `NotificationServer`
- Configure capabilities (`bodySupported`, `actionsSupported`, `bodyImagesSupported`, `bodyMarkupSupported`)
- Manage `trackedNotifications` list
- Handle new notification signal

### 2. `shell/notifications/NotificationToast.qml`
- Individual toast popup component
- Display: app icon, app name, summary, body (optional image)
- Auto-dismiss timer based on `expireTimeout` (default 5s)
- Click to dismiss, action buttons if present
- Animations for show/hide

### 3. `shell/notifications/ToastManager.qml`
- Manages stacking of multiple toasts
- Position configurable (top-right by default)
- Queue management (max visible toasts)
- Uses `PanelWindow` with `WlrLayerShell`

### 4. `shell/notifications/NotificationCenter.qml`
- Popup panel showing all tracked notifications
- Clear all / dismiss individual buttons
- Action support
- Opened from bar widget or keybind

### 5. `shell/notifications/widgets/NotificationItem.qml`
- Reusable notification display component
- Used by both Toast and NotificationCenter
- Supports: image, app icon/name, summary, body, urgency styling, actions

### 6. `shell/bar/widgets/Notifications.qml` (optional)
- Bar widget showing notification count
- Opens NotificationCenter on click

## Implementation Details

### NotificationServer Pattern
```qml
pragma Singleton
import Quickshell
import Quickshell.Services.Notifications

Singleton {
    property NotificationServer server: NotificationServer {
        bodySupported: true
        bodyMarkupSupported: true
        bodyImagesSupported: true
        actionsSupported: true
        persistenceSupported: true
        
        onNotification: (notification) => {
            notification.tracked = !notification.transient
        }
    }
}
```

### Toast Positioning
- Use `WlrLayerShell.layer: WlrLayer.Overlay`
- Anchor to screen edge based on config
- Stack vertically with spacing

### Styling
- Use existing `Theme.qml` colors
- Urgency-based border colors (low/normal/critical)
- Match existing bar/launcher styling
